# Apartment Search Preferences Example

Use this file to describe your own search preferences before converting them into a structured profile.

Example:

- Move-in: July 1
- Lease length: 12 months
- Bedrooms: 2+
- Bathrooms: 1+
- Budget: target and stretch ranges
- Laundry: in-unit or in-building preferred
- Commute destination: your workplace or school address
- Preferred neighborhoods: list neighborhoods in priority order
- Dealbreakers: list hard rejections
- Qualitative preferences: natural light, layout, kitchen, hosting space, building amenities, noise, security

Do not include private information in this example file.
